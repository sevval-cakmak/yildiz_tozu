# ✨ Yıldız Tozu

## 🇹🇷 Türkçe

Yıldız Tozu, Django ile geliştirdiğim kişisel bir web uygulamasıdır. Sevdiğim müzikleri, filmleri, kitapları ve bana ilham veren içerikleri modern ve sade bir arayüzle sunmayı amaçlamaktadır.

İçerikler, Django Admin paneli üzerinden kolayca yönetilebilmekte ve dinamik olarak kullanıcıya sunulmaktadır.

### Özellikler

- Kişisel içerik sayfaları
- Django Admin ile içerik yönetimi
- Responsive ve minimalist arayüz
- Dinamik içerik yapısı

### Teknolojiler

- Django
- Python
- HTML
- CSS
- JavaScript
- SQLite

---

## 🇬🇧 English

Yıldız Tozu is a personal web application built with Django. It brings together my favorite music, movies, books, and inspiring content through a clean and modern interface.

Content can be easily managed via the Django Admin panel, providing a dynamic and maintainable structure.

### Features

- Personal content pages
- Content management with Django Admin
- Responsive and minimalist interface
- Dynamic content structure

### Technologies

- Django
- Python
- HTML
- CSS
- JavaScript
- SQLite
